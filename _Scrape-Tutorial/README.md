Code here is based on the web scraping/Node.js tutorial found here: https://www.digitalocean.com/community/tutorials/how-to-scrape-a-website-with-node-js

Prerequisites for this project:
1. Node.js
2. nom
3. Cheerio
4. Axios
5. Express
6. Lodash
